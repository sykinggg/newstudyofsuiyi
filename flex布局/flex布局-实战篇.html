模板描述

div元素（代表骰子的一个面）是Flex容器，span元素（代表一个点）是Flex项目

如果有多个项目，就要添加多个span元素，以此类推

单项目

水平方向居左,垂直方向居上

Flex布局默认就是首行左对齐，所以一行代码就够了

解决方法

			.box {
				display: flex;
			}

水平方向居中,垂直方向居上

设置项目的对齐方式，就能实现居中对齐和右对齐

解决方法

			.box {
				display: flex;
				justify-content: center;
			}

水平方向居右,垂直方向居上

解决方法

			.box {
				display: flex;
				justify-content: flex-end;
			}

水平方向居左,垂直方向居中

设置交叉轴对齐方式，可以垂直移动主轴

解决方法

			.box {
				display: flex;
				align-items: center;
			}

水平方向居中,垂直方向居中

解决方法

			.box {
				display: flex;
				justify-content: center;
				align-items: center;
			}

水平方向居中,垂直方向居下

解决方法

			.box {
				display: flex;
				justify-content: center;
				align-items: flex-end;
			}

水平方向居右,垂直方向居下

解决方法

双项目

水平居左,垂直居上;水平居右,垂直居上

解决方法

			.box {
				display: flex;
				justify-content: space-between;
			}

水平居左,垂直居上;水平居左,垂直居下

解决方法

			.box {
				display: flex;
				flex-direction: column;
				justify-content: space-between;
			}

水平居中,垂直居上;水平居中,垂直居下

解决方法

			.box {
				display: flex;
				flex-direction: column;
				justify-content: space-between;
				align-items: center
			}

水平居右,垂直居上;水平居右,垂直居下

解决方法

			.box {
				display: flex;
				flex-direction: column;
				justify-content: space-between;
				align-items: flex-end;
			}

水平居左,垂直居上;水平居中,垂直居中

解决方法

			.box {
				display: flex;
			}

			.item:nth-child(2) {
				aling-self: center;
			}

水平居左,垂直居上;水平居右,垂直居下

解决方法

			.box {
				display: flex;
				justify-content-space-between;
			}

			.item:nth-child(2) {
				align-self: flex-end;
			}

三项目

水平左,垂直上;水平中,垂直中;水平右,垂直下

解决方法

			.box {
				display: flex;
			}

			.item:nth-child(2) {
				align-self: center;
			}

			.item:nth-child(3) {
				align-self: flex-end;
			}
