javascript的eval()

eval()的参数是一个字符串，eval()会将传入的字符串当做代码来执行

概念描述

在Javascript中eval()并不是唯一可以执行Javascript字符串的函数

使用Function构造函数亦可以做到这一点

setTimeout()和setInterval()也可以

尽管在Javascript类库中eval()，可能会经常用到（通常和JSON操作有关）

另外三种用法即使也非常罕见

一个通用的原则是，禁止使用Function，并且只在别无他法时使用，eval()

setTimeout和setInterval也是可以使用的，但不要用字符串形式而要用函数
