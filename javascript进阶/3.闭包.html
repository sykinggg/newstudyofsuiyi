闭包是指函数有自由独立的变量。换句话说，定义在闭包中的函数可以“记忆”它创建时候的环境

词法作用域

代码描述

函数 init() 创建了一个局部变量 name，然后定义了名为 displayName() 的函数

displayName() 是一个内部函数——定义于 init() 之内且仅在该函数体内可用

displayName() 没有任何自己的局部变量，然而它可以访问到外部函数的变量，即可以使用父函数中声明的 name 变量
