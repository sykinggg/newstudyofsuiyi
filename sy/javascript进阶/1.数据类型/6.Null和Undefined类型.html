Null和Undefined类型

Null和Undefined类型都只有唯一的特殊值,即null和undefined

注意:

null是关键字而undefined不是

注意:

JavaScript把两者值定义为相等的

JavaScript把两者概念定义为不等的

概念描述:

undefined表示更深层次的“空值”，它是变量的一种取值，表明变量没有初始化

null 则用于表示尚未存在的对象

如果函数或方法要返回的是对象，那么找不到该对象时，返回的通常是 null

注意:(也可以)

认为undefined是表示系统级的、出乎意料的或类似错误的值的“空值”

null表示程序级的、意料之中的值的“空值”

注意:(使用中)

如果你想吧两者赋值给变量或者属性等，最好的选择是null
