web app 之 rem

rem（font size of the root element）是指相对于根元素的字体大小的单位

em（font size of the element）是指相对于父元素的字体大小的单位

它们之间其实很相似，只不过一个计算的规则是依赖根元素一个是依赖父元素计算

流式布局

  • 页面布局的时候都是通过百分比来定义宽度
  • 高度大都是用px来固定住

流式布局的缺点

在大屏幕的手机下显示效果会变成有些页面元素宽度被拉的很长

但是高度还是和原来一样，实际显示非常的不协调

屏幕适配布局

  • 固定宽度做法
  • 响应式做法

  • 设置viewport进行缩放

    这个方法简单粗暴，又高效但是会模糊

最佳解决方案

rem能等比例适配所有屏幕

改变html的font-size的值

可以计算出1px多少rem

120px = 6rem * 20px(根元素设置大值)

240px = 6rem * 40px(根元素设置大值)

推算出

  • 10px = 1rem 在根元素（font-size = 10px的时候）
  • 20px = 1rem 在根元素（font-size = 20px的时候）
  • 40px = 1rem 在根元素（font-size = 40px的时候）

表格描述

表格蓝色一列是Demo3中页面的尺寸，页面是以640的宽度去切的

384/640 = 0.6，384是640的0.6倍，所以384页面宽度下的font-size也等于它的0.6倍

这时384的font-size就等于12px
