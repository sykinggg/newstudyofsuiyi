function 语句

function 语句用于定义函数对象 
      function fd() {
        //do sth.
        return true;
      }
用function创建函数对象一般称之为函数声明,与之对相应的是函数表达式 
      函数表达式是声明一个变量然后把一个匿名函数赋值给这个变量
      var fa = function() {
        // do sth.
      }

注意: 函数声明和函数表达式之间的区别

最主要的是:函数声明会被预先处理或者说是函数前置也就是说在函数声明的前面来 调用该函数声明也是可以的(可以被调用成功);但是函数表达式就不行

for ... in语句

      使用语法:
      var p;
      var obj = {x: 1, y: 2};
      for(p in obj) {
        console.log(p);
      }

      使用时三点注意:
        1.顺序不确定(依赖于引擎的实现)
        2.enumerable为false时不会出现(属性描述器)
        3.for in对象属性时受原型链影响(注意如果obj的原型的enumerable为true也会被for in循环出来)

switch 条件分支语句

      var val = 2;
      switch(val) {
        case 1:
          console.log(1);
          break;
        case 2:
          console.log(2);
          break;
        case 3:
          console.log(0);
          break;
      }

      /*
        运行结果
          2
      */


      switch(val) {
        case 1:
          console.log(1);
        case 2:
          console.log(2);
        default:
          console.log(0);
      }
      /*
        运行结果
          2
          0
      */
      执行流程: 没有break则继续往下执行


      switch(val) {
        case 1:
        case 2:
        case 3:
          console.log(123);
          break;
        case 4:
        case 5:
          console.log(45);
          break;
        default:
          console.log(0);
      }
      /*
        运行结果:
          123
      */

with 语句

可以修改当前的作用域 
      基本语法:
        with({x: 1}) {
          console.log(x);
        }
      优点: 不用写对象.x
      with(document.forms[0]) {
        console.log(name.value);
      }
      其实就是代替
        console.log(document.forms[0].name.value);
      也可以用
        var form = document.forms[0];
        console.log(form.name.value);
      js现在基本上不使用with
        1.让js引擎优化更难
        2.可读性变差
        3.可被变量定义代替
        4.严格模式下禁用
      总结:如果要定义深层访问则可用定义变量的方式来代替with
