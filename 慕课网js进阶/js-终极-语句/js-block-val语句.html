块语句 block

块语句常用于组合多个语句,块语句用一对花括号定义

语法格式

        语法 {
          语句1;
          语句2;
          ...
          语句n;
        }
    
      注意:很少单独使用一般与if什么的语句连用
      if(true) {
        console.log("hi");
      }

注意:没有块级作用域

代码描述

      for(var i = 0; i < 10; i++) {
        var str = "hi";
        console.log(str);
      }
      var j = 0
      for(; j < 10; j++) {
        var str = "hi";
        console.log(str);
      }
      注意上面的两段代码在作用域的角度来说是没有任何区别的

注意:有函数的作用域

      function foo() {
        var a = 1;
        console.log(a);
      }
      foo();
      console.log(typeof a);
      数出结果: undefined
      也就是有函数作用域但没有块级作用域!

var 声明语句

定义一个变量

      var a = 1;

注意:var作用域

      var a = b = 1;
      注意:b是隐式创建的全局变量
      function foo() {
        var a = b = 1;
      }
      foo();
      console.log(typeof a);
      结果为 undefined
      console.log(typeof b);
      结果为 number

解决问题的方法

      var a = 1, b = 1;
