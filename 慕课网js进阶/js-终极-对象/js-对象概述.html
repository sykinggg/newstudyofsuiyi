基本概念

      对象中包含一系列属性,这些属性是无序的.
      每个属性都有一个字符串key和对应的value.
          var obj = {x: 1, y: 2};
          obj.x; //1
          obj.y; //2

属性的key值是字符串

对象的结构

js中的对象的属性是动态的 
      内置的属性
            writable
            encumerable
            configurable
            value
            get/set
      以及原型__proto__
            class: 来表明对象的分类
            extensible: 来表明是否可以新增属性
