js的值

JavaScript的语法定义了两种类型的值：固定值和变量值

固定值称为文字。变量值被称为变量

js的字

数字包含整型和浮点型

js变量

在编程语言中，变量是用来存储数据值的

JavaScript使用var关键字声明变量

一个平等的符号是用来分配值的变量

js的规则

JavaScript使用赋值运算符（=）给变量赋值

JavaScript使用算术运算符（+ - * /）来计算值

JavaScript表达式

表达式是值、变量和运算符的组合，计算其值

JavaScript的关键词

JavaScript语句通常从一个关键字识别JavaScript执行动作

Keyword Description
break 终止一个开关或一个循环
continue 跳出一个循环，并开始在顶部
debugger 停止JavaScript的执行，并要求（如果有）的调试功能
do ... while 执行一个语句块，然后重复该块，而一个条件是真的
for 标记一个语句块被执行，只要条件是真的
function 声明一个函数
if ... else 标志着一个语句块被执行，取决于一个条件
return 中断并退出函数
switch 标志着一个语句块被执行，根据不同的情况
try ... catch 对语句块进行错误处理
var 声明一个变量

注意:JavaScript的关键词是保留字。保留字不能用作变量的名称。
