Animations

在模板中用[@animationName]语法来把它附加到一个或多个元素上

*(通配符)状态

*(通配符)状态匹配任何动画状态

  • 当该元素的状态从active变成任何其它状态时，active => *转场都会生效
  • 当在任意两个状态之间切换时，* => *转场都会生效

void状态

  • 表示元素没有被附加到视图。这种情况可能是由于它尚未被添加进来或者已经被移除了
  • 当一个元素离开视图时，* => void转场就会生效，而不管它在离场以前是什么状态

*通配符状态也能匹配void

使用void和*状态，可以定义元素进场与离场时的转场动画
  • 进场：void => *
  • 离场：* => void

动画时间线

  • 持续时间(duration)
  • 延迟(delay)
  • 缓动(easing)函数
