@HostListener装饰器引用属性型指令的宿主元素，在这个例子中就是p

可以通过标准的JavaScript方式手动给宿主 DOM 元素附加一个事件监听器
  • 必须正确的书写事件监听器
  • 当指令被销毁的时候，必须拆卸事件监听器，否则会导致内存泄露
  • 必须直接和 DOM API 打交道，应该避免这样做
